# HeadHunter Clone API (FastAPI)

Асинхронное API для управления данными сотрудников и вакансиями. Проект разработан с акцентомр на современную архитектуу, асинхронную работу с базой данных и автоматизированное тестирование.

## Стек технологий

* **Язык:** Python 3.12
* **Фреймворк:** FastAPI (Async)
* **ORM:** SQLAlchemy 2.0 (Async)
* **База данных:** PostgreSQL
* **Тестирование:** Pytest + HTTPX
* **Окружение:** Docker & Docker Compose
* **Валидация:** Pydantic v2
* **Миграции:** Alembic
* **Контроль зависимостей:** Poetry


## Ключевые особенности

* **Полная асинхронность:** Все операции с БД (CRUD, фильтрация) выполняются асинхронно.
* **Интеграционное тестирование:** Реализовано базовое интерграционное тестирование основных операций.
* **Docker, docker compose:** Проект полностью контейнеризирован и готов к развертыванию одной командой.
* **Скрипты для автоматизации:** скрипт для создания пакетов, обрабатывающих crud и маршрутизацию; скрипт для создания администраторов
* **Использование паттернов проектирования:** в проекте использовались такие паттерны проектирования как строитель, стратегия, посредник
* **Использование принципов SOLID**
* **Кастомная логика фильтров:** была разработана кастомная логика для обработки фильтров через query параметры
* **Работа с переменными окружения:** использование pydantic settings и .env файлов
* **Bearer аутентификация с использованием JWT токенов**
* **Авторизация:** роли админа и пользователя

## Как запустить проект

1. **Клонируйте репозиторий:**
2. **Создайте файлы с переменными окружения (настройки подключения к базе данных)**
3. **Пропишите в терминал bash команду: docker compose up**
4. **Примените миграции с помощью следующей команды в терминале: docker compose exec alembic upgrade head**
5. **Перейдите в браузере по адресу 0.0.0.0:5000 (если не указали свой хост и порт)**
